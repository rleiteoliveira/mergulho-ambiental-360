# Teste no Meta Quest 3

## Objetivo

Validar o que não pode ser confirmado no Editor ou na web demo:

- conforto;
- legibilidade;
- input real;
- performance;
- orientação do vídeo;
- operação em exposição.

## Preparação

- Quest 3 em Developer Mode.
- Android Build Support instalado no Unity.
- OpenXR configurado.
- APK gerado a partir de `unity-app/`.
- Pelo menos um vídeo 360 real, curto e autorizado.

## Execução

1. Instalar APK.
2. Abrir app.
3. Conferir menu.
4. Selecionar vídeo.
5. Assistir por alguns minutos.
6. Testar pause, voltar e próximo.
7. Validar operação por professor/monitor.
8. Registrar problemas no checklist.

## Observação

Sem o Quest 3 real, não prometer conforto, performance, legibilidade infantil ou compatibilidade final.
