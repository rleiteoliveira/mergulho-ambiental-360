# PoC Goals

## Validar

- Se app nativo Unity é realmente necessário para exposição.
- Se web demo pode apoiar demonstração, catálogo ou alinhamento com cliente.
- Como organizar vídeos 360 por catálogo.
- Como seria o fluxo mínimo menu + player.
- Quais decisões dependem do Quest 3 real.
- Quais perguntas precisam ser respondidas pelo cliente.

## Não fazer agora

- Backend.
- Multiplayer.
- Jogo completo.
- Quiz.
- Coleta de dados.
- Publicação em loja.
- Uso de vídeos de terceiros sem autorização.
- Promessa de compatibilidade final sem teste no headset.
